# IT Support Ticket Simulation

This section shows IT support tickets based on common home IT problems.

Each ticket includes:
- Problem  
- Steps to fix it  
- Solution  

---

## Ticket 001 - No Internet Connection

**Problem:**
User has no internet connection on the laptop.

**Steps to Fix:**
- Checked Wi-Fi connection  
- Restarted router and modem  
- Turned off airplane mode  
- Reconnected to Wi-Fi  
- Restarted the laptop  

**Solution:**
Internet worked again after restarting router and reconnecting to Wi-Fi.

---

## Ticket 002 - Slow Wi-Fi Connection

**Problem:**
Internet is slow when browsing or streaming.

**Steps to Fix:**
- Checked number of connected devices  
- Removed unused devices from network  
- Restarted router  
- Moved closer to router  
- Closed background apps  

**Solution:**
Internet speed improved after reducing connected devices and restarting router.

---

## Ticket 005 - Printer Not Printing

**Problem:**

**Steps to Fix:**

**Solution:**
Printer worked again after fixing connection and queue issues.
